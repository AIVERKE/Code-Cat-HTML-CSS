Actividad Práctica: Listas y Tablas

Descripción de la actividad

Crear un documento HTML que contenga:

  1. Una lista de elementos favoritos
    • Puede ser una lista desordenada (<ul>) o una lista ordenada (<ol>).
    • Debe incluir al menos tres elementos (<li>) que representen gustos o preferencias (comidas, películas o hobbies).
  2. Una tabla sencilla
    • La tabla debe tener dos filas y dos columnas.
    • Una columna puede ser “Nombre” y la otra “Elemento favorito”.
    • Incluir un encabezado (<th>) en la primera fila para identificar las columnas.

Ejemplo de lista y tabla

<!-- Lista de elementos favoritos -->
<ul>
  <li>Papa a la Huancaina</li>
  <li>Tocar Guitarra</li>
  <li>Programación</li>
</ul>

<!-- Tabla sencilla -->
<table border="1">
  <tr>
    <th>Nombre</th>
    <th>Elemento favorito</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>Pizza</td>
  </tr>
  <tr>
    <td>Diego</td>
    <td>Programación</td>
  </tr>
</table>

Instrucciones

  1. Abrir un editor de texto y crear un archivo HTML nuevo (por ejemplo, listas_tablas.html).
  2. Escribir la estructura básica de HTML (<!DOCTYPE html>, <html>, <head>, <title>, <body>).
  3. Crear la lista de elementos favoritos usando <ul> o <ol> y <li> para cada elemento.
  4. Construir la tabla con etiquetas <table>, <tr>, <th> y <td> siguiendo la estructura mínima indicada.
  5. Abrir el archivo en un navegador para verificar que la lista y la tabla se muestran correctamente.
