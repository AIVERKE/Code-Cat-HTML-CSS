Actividad Práctica: Scripts e Iframe

Descripción de la actividad

Crear un documento HTML que contenga:

  • Un script simple que muestre una alerta al cargar la página.
  • Un iframe para embeber un recurso externo, como un mapa o un video.

Ejemplo de código

<!DOCTYPE html>
        <html lang="es">
        <head>
        <meta charset="UTF-8">
        <title>Scripts e Iframe</title>
        <script>
            window.onload = function() {
            alert("Mensaje de bienvenida");
            };
        </script>
        </head>
        <body>

        <h1>Actividad de Scripts e Iframe</h1>

        <p>Aquí se puede insertar un recurso externo embebido:</p>

        <!-- Iframe de ejemplo -->
        <iframe src="#" width="560" height="315"></iframe>

        </body>
        </html>

Instrucciones

  1. Abrir un editor de texto y crear un archivo HTML nuevo (por ejemplo, scripts_iframe.html).
  2. Escribir la estructura básica de HTML (<!DOCTYPE html>, <html>, <head>, <title>, <body>).
  3. Agregar un script que muestre un mensaje de alerta al cargar la página.
  4. Insertar un iframe con un recurso externo (video o mapa).
  5. Abrir el archivo en un navegador para verificar que la alerta aparece y que el recurso externo se muestra correctamente.
