Actividad Práctica Final: Desafío 2

Descripción de la actividad

Crear un documento HTML que contenga los siguientes elementos:

  1. Formulario de contacto completo con validación
    • Campos de texto, email, textarea para mensaje y botón de envío.
    • Validación HTML5 usando required y tipos apropiados (email, text).
  2. Estructura semántica completa
    • Incluir <header>, <main> con <section> y <article>, y <footer>.
  3. Elemento <details>
    • Contener información adicional que el usuario pueda desplegar.
  4. Iframe embebido
    • Mostrar un mapa o video externo.
  5. Script básico de interacción
    • Mostrar un mensaje o alerta cuando el usuario interactúe con el formulario.

Instrucciones

  1. Abrir un editor de texto y crear un archivo HTML nuevo (por ejemplo, desafio2.html).
  2. Escribir la estructura básica de HTML (<!DOCTYPE html>, <html>, <head>, <title>, <body>).
  3. Incorporar todos los elementos indicados en la descripción, asegurándose de usar etiquetas semánticas y validaciones correctas.
  4. Abrir el archivo en un navegador para verificar que todos los elementos funcionan correctamente, incluyendo el formulario, <details>, iframe y script.
