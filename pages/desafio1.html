Actividad Práctica Final

Descripción de la actividad

Crear un documento HTML que contenga los siguientes elementos:

  1. Título y párrafo
    • Usar <h1> para el título principal.
    • Incluir al menos un <p> con texto descriptivo o informativo.
  2. Texto con formato y semántica
    • Aplicar etiquetas de formato visual como <b>, <i>, <u>, <strong>, <em>.
    • Utilizar etiquetas semánticas como <abbr>, <cite> o <code> para enriquecer el contenido.
  3. Una lista
    • Crear una lista ordenada <ol> o desordenada <ul> con al menos tres elementos <li>.
  4. Tabla con dos elementos
    • Construir una tabla <table> con dos filas y dos columnas.
    • Incluir encabezados <th> y datos <td> que representen información simple y clara.
  5. Imagen con pie de foto
    • Insertar una imagen mediante <figure> y <img> con atributo alt.
    • Añadir un <figcaption> describiendo la imagen.
  6. Frase con símbolos o emojis
    • Incluir una frase que contenga entidades HTML (&lt;, &gt;, &amp;) o códigos numéricos para símbolos y emojis (&#9829;, &#128512;).

Instrucciones

  1. Abrir un editor de texto y crear un archivo HTML nuevo (por ejemplo, reto_final.html).
  2. Escribir la estructura básica de HTML (<!DOCTYPE html>, <html>, <head>, <title>, <body>).
  3. Incorporar todos los elementos indicados en la descripción, asegurándose de utilizar etiquetas correctas y semánticas.
  4. Abrir el archivo en un navegador para verificar que todos los elementos se muestran correctamente y funcionan como se espera.
