Introducción a CSS

CSS (Cascading Style Sheets) se utiliza para dar estilo y formato a las páginas web, separando la presentación del contenido.

Formas de aplicar CSS

1. Inline (directamente en la etiqueta)

Ejemplo:

<p style="color: red; font-size: 18px;">Texto en rojo y tamaño 18px</p>

2. Interno (en la misma página dentro de <style>)

<style>
p {
    color: blue;
    font-size: 16px;
}
</style>

<p>Texto azul con CSS interno</p>

3. Externo (archivo .css)

<link rel="stylesheet" href="estilos.css">

Sintaxis básica

selector {
    propiedad: valor;
}

Selectores principales

/* Selector por elemento */
p {
    color: green;
}

/* Selector por clase */
.clase-ejemplo {
    background-color: yellow;
}

/* Selector por ID */
#id-ejemplo {
    font-size: 20px;
}

Propiedades básicas

p {
    color: red;              /* Color del texto */
    font-size: 18px;         /* Tamaño de la fuente */
    background-color: lightgray; /* Color de fondo */
    margin: 10px;            /* Margen exterior */
    padding: 5px;            /* Espacio interior */
}

Resumen

  • Aplicar CSS de tres formas: inline (en la etiqueta), interno (dentro de <style>) y externo (archivo .css).
  • La sintaxis básica incluye un selector y un bloque de propiedades con valores.
  • Usar selectores principales: por elemento, por clase (.) y por ID (#).
  • Propiedades básicas para estilo: color, font-size, background-color, margin y padding.
  • Combinar selectores y propiedades para dar formato y resaltar elementos de la página.
