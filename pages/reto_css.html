Actividad Práctica: CSS Básico en Reto Final

Descripción de la actividad

Aplicar estilos básicos a los elementos de un reto final HTML para visualizar cómo CSS puede mejorar la presentación:

  • Modificar colores de fondo y texto de header, footer y nav.
  • Agregar márgenes, padding y bordes redondeados a formularios y elementos principales.
  • Estilizar botones, inputs y textareas para mejorar su apariencia.
  • Usar hover para efectos interactivos en botones y enlaces.

Instrucciones

  1. Abrir un editor de texto y crear un archivo HTML nuevo (por ejemplo, actividad_css.html).
  2. Agregar la estructura básica de HTML y algunos elementos de ejemplo como formularios, botones y detalles.
  3. Aplicar estilos CSS internos para modificar colores, márgenes, padding, bordes y efectos hover.
  4. Abrir el archivo en un navegador para ver cómo los estilos cambian la apariencia y facilitan la lectura.
