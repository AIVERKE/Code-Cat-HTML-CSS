Entidades, Símbolos y Emojis en HTML

En HTML existen caracteres que no pueden escribirse directamente porque se confundirían con la sintaxis del lenguaje. Para representarlos se utilizan entidades HTML y códigos numéricos para símbolos y emojis.

Entidades HTML básicas

Las entidades HTML comienzan con & y terminan con ;. Los navegadores las interpretan correctamente.

<p>2 < 5 y 5 > 2</p>
<p>Uso del símbolo & para unir palabras.</p>

Símbolos y emojis mediante códigos numéricos

HTML permite usar códigos numéricos que comienzan con &# y terminan con ;.

<p>Te envío un corazón: ♥</p>
<p>Sonríe: 😀</p>

Aplicaciones prácticas

  • Representar caracteres especiales sin interferir con etiquetas HTML.
  • Incluir símbolos y emojis para enriquecer la presentación de información.
  • Mejorar la accesibilidad y compatibilidad entre navegadores.

Resumen

  1. Se usan entidades HTML para caracteres reservados: <, >, &.
  2. Se usan códigos numéricos para mostrar símbolos y emojis: ♥, 😀.
  3. Permite mostrar contenido correctamente en todos los navegadores y plataformas.
