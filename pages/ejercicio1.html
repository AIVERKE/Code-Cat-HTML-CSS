Actividad Práctica: Plantilla mínima en HTML

Descripción de la actividad

Crear un documento HTML que contenga:

  • Declaración del tipo de documento (<!DOCTYPE html>) para indicar HTML5.
  • Etiqueta raíz <html> con el atributo lang="es".
  • Encabezado <head> que incluya:
    • <meta charset="UTF-8"> para codificación de caracteres.
    • <title> con un nombre representativo para la página.
  • Cuerpo del documento <body> que contenga:
    • Un encabezado principal (<h1>) con el título de la página.
    • Un párrafo (<p>) con texto de ejemplo.
    • Un comentario HTML explicativo (<!-- Comentario -->) que no se mostrará en el navegador.

Ejemplo de plantilla mínima

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi primera página</title>
</head>
<body>
  <h1>Bienvenidos a HTML</h1>
  <p>Este es un párrafo de ejemplo.</p>
  <!-- Este es un comentario que no se muestra en el navegador -->
</body>
</html>

Instrucciones

  1. Abrir un editor de texto (Visual Studio Code, Sublime Text o Notepad++).
  2. Crear un nuevo archivo y guardarlo con extensión .html (por ejemplo, plantilla.html).
  3. Escribir el código HTML siguiendo la estructura indicada.
  4. Abrir el archivo en un navegador para verificar que se visualiza correctamente el encabezado y el párrafo, asegurándose de que el comentario no se muestre.
