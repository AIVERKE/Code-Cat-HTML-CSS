Fundamentos y Estructura HTML

Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para la creación de páginas web. Su función principal es estructurar el contenido mediante etiquetas (tags).

  • No es un lenguaje de programación, sino un lenguaje de marcado.
  • Define la estructura y semántica del contenido.
  • Trabaja con CSS y JavaScript.

Documentos HTML: Doctype y etiquetas raíz

Todo documento comienza con el DOCTYPE que indica la versión del lenguaje:

<!DOCTYPE html>

Luego se define la etiqueta raíz:

<html lang="es"> ... </html>

Elementos del encabezado

Dentro del <head> se colocan elementos no visibles directamente como:

  • <title> → título de la página.
  • <meta> → metadatos de la página.
<head>
   <meta charset="UTF-8">
   <title>Mi primera página</title>
   <meta name="author" content="Diego Gutiérrez">
   <meta name="description" content="Ejemplo introductorio de HTML">
</head>

Cuerpo del documento

<body>
   <h1>Bienvenidos a HTML</h1>
   <p>Este es mi primer párrafo.</p>
</body>

Encabezados

<h1>Encabezado principal</h1>
<h2>Subtema</h2>
<h3>Detalle específico</h3>

Párrafos y saltos de línea

<p>Este es un párrafo.</p>
<p>Este es otro párrafo.</p>
<p>Línea uno.<br>Línea dos.</p>

Comentarios

<!-- Este es un comentario -->
<p>Texto visible</p>

Resumen

<!DOCTYPE html>
<html lang="es">
<head>
   <meta charset="UTF-8">
   <title>Ejemplo de HTML</title>
</head>
<body>
   <h1>Encabezado principal</h1>
   <p>Este es un párrafo de ejemplo.</p>
   <!-- Comentario -->
</body>
</html>
